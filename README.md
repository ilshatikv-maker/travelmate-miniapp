# TravelMate Mini App

Мини-приложение для Telegram "TravelMate" - планировщик путешествий.

## Функции
- Создание поездок
- Управление бюджетом
- Список поездок
- Интеграция с Telegram Web App API

## Как запустить
1. Открой в Telegram: `https://t.me/travelmate26_bot?startapp=test`
2. Или через меню бота

## Технологии
- HTML5
- CSS3
- JavaScript
- Telegram Web App API
